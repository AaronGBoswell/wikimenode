Index.html:
	Get Tweets button:
		Display some information about all tweets
	Get Users button:
		Display some information about all users
	Get External Links button:
		Display all external links, by tweet ID
	Search Tweets button:
		Display more information about the tweet, who's ID is in the search field
	Search User button:
		Display more information about the user, who's screen name is in the search field

	GET /data/tweets
	GET /data/tweets/<id>
	GET /data/users
